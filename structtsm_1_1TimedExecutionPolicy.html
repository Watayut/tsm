Detailed Description

template<typename StateType, template< typename > class TimerType, typename DurationType>
struct tsm::TimedExecutionPolicy< StateType, TimerType, DurationType >

The policy for timed event processing. This Policy class works with a Timer type. A callback from this policy is invoked from the Timer every time a preset time period expires.
