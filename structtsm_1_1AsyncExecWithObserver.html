Detailed Description

template<typename StateType, typename Observer>
struct tsm::AsyncExecWithObserver< StateType, Observer >

Another asynchronous execution policy. The only difference with above is that an Observer's notify method will be invoked at the end of processing each event - specifically, right before the blocking wait for the next event.
