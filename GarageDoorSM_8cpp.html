GarageDoorHsm is the state machine definition. It has knowledge of the Hsm hierarchy, its states, events and sub-Hsms if any. The relationships between Hsms (parentHsm_) is also setup here. Mix the Async observer and the GarageDoor Hsm to get an asynchronous garage door state machine that notifies a listener at the end of processing each event.
